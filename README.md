# OpenW3D Docs

A place where we gather knowledge around the W3D engine, created by EA games. Feel free to contribute.

## File Formats

* [.w3d](file-formats/w3d/) - 3D model file, used to store meshes, animations, and other model-related data
* [.mix](file-formats/mix/) - archive container format used for packaged game data
* [.ddb](file-formats/ddb/) - chunk-based binary data format
* [.lvl](file-formats/lvl/) - chunk-based level data format
* [.lsd](file-formats/lsd/) - chunk-based binary data format
* [.ldd](file-formats/ldd/) - chunk-based dynamic save data format
* [.cdb](file-formats/cdb/) - chunk-based conversation database format
* [.tdb](file-formats/tdb/) - chunk-based translation database format
